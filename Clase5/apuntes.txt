SOBRECARGA DE OPERADORES

(visibilidad) static (retorno) operator (operador) (dato, dato)


ej:

public static bool operator == (Tinta unaTinta, Tinta otraTinta)


este metodo se llama "==". Adentro, va un if que compara ambas tintas y retorna true o false segun sean iguales o no


-para llamarlo y mostrar por pantalla:


console.write(unaTinta == otraTinta)
